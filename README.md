# Rock Paper Scissors 🎮

A simple command-line Rock Paper Scissors game built with Python.

## 📌 Description
This is a beginner-level Python project where the user plays Rock, Paper, Scissor against the computer.  
The computer randomly selects its choice and the result is shown based on the game rules.

## 🚀 How the Game Works
- The computer randomly chooses between Rock, Paper, and Scissor
- The user enters their choice
- The program compares both choices and prints the result
- The game continues until the user types `Exit`

## ▶️ How to Run
1. Make sure Python is installed on your system
2. Run the file using the command below:

```bash
python Rock_Paper_Scissors.py
