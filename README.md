# Rock Paper Scissors 🎮

A command-line Rock Paper Scissors game built with Python.

## 📌 Description
This is a beginner-level Python project where the user plays Rock, Paper, Scissor against the computer. The computer randomly selects its choice, and the game keeps running until the user decides to quit.

## 🚀 Features
- Play against the computer
- Random computer choices
- Win / Lose / Draw detection
- Input validation
- Score tracking
- Statistics tracking
- Exit the game anytime

## 🎯 Scoring System
- Win = 3 points
- Draw = 1 point
- Loss = 0 points

## 🚀 How the Game Works
- The computer randomly chooses between Rock, Paper, and Scissor
- The user enters their choice
- The program compares both choices and displays the result
- Wins, losses, draws, and points are tracked during the game
- Statistics are displayed when the user exits

## ▶️ How to Run
1. Make sure Python is installed on your system
2. Run the file using the command below:

```bash
python Rock_Paper_Scissors.py
```

## ⌨️ Controls
- Type `Rock`, `Paper`, or `Scissor` to play
- Type `Exit` to quit the game

## 📚 Concepts Used
- Python loops (`while`)
- Conditional statements (`if`, `elif`, `else`)
- Random module
- User input handling
- Variables and counters
- Basic game logic
